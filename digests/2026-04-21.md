### Andrej Karpathy — "LLM Knowledge Bases"
Karpathy shared (early April) that he's shifted from using AI to write code to using it as a "second brain." His approach: dump raw research into a folder, point an LLM at it, and let it build an interlinked wiki autonomously. His personal wiki has grown to ~100 articles and 400K words. The open-source implementation hit 5,000+ GitHub stars within days. This represents a shift from retrieval to compilation — the LLM synthesizes rather than retrieves.
